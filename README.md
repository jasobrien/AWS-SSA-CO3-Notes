# AWS Solutions Architect Associate (SAA-C03) — Notes Index

These notes are organized like a small “book” (chapters by topic). They focus on high-yield decision points, common exam traps, and architecture patterns.

## How to use

- Start with **Foundations**, then **Networking** and **Security/Identity**.
- Use the **Practice** questions folder to test recall.
- When a question asks “MOST cost-effective / MOST resilient / MOST secure”, jump to the relevant chapter and scan the “Exam tell” bullets.
